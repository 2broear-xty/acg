# ACGN 卡片页面
一个图文卡片式静态页面，可用作于（ACGN）动漫、漫画、音乐、电影及文字阅读书籍收藏等页面。
## 预览地址
[预览 Demo](https://app.2broear.com/acg/)
#### 主要文件
acg.js/acg.json/acg.css（index.html）
#### 依赖项
jquery.js

# 新增&修改
#### 已存在对象
通过 acg.json 添加修改已存在条目（在对应 json 文件中修改对象）
#### 新增对象
通过 acg.js 新增 jsonObj 对象，再通过 acg.json 新增同名 json 对象
