# 静态卡片页面
一个静态页面，可用作于电影收藏、音乐收藏、阅读书籍记录等页面。
## 预览地址
[预览 Demo](https://app.2broear.com/acg/)
### 主要文件
acg.js/acg.json
### 其他文件
acg.css/index.html
### 依赖项
jquery.js/~~lazyload.js~~

# 新增&修改
通过 acg.json 添加修改已存在条目（在对应 json 文件中修改对象即可）
通过 acg.js/acg.json 新增条目（在 acg.js 新增对象后需要同步在 acg.json 新增相同类名数据否则会抛出错误）
